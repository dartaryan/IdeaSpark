# IdeaSpark

BMAD (Build, Manage, and Deploy) Platform project.

## Overview

This project contains the BMAD Core Platform with modules for agents, workflows, and tools.

## Structure

- `_bmad/` - BMAD platform core and modules
  - `core/` - Core platform functionality
  - `bmb/` - Build module builder
  - `bmm/` - Build module modules
- `_bmad-output/` - Generated output files (excluded from git)
- `user-data/` - User-specific data and artifacts

## Getting Started

See the BMAD documentation in `_bmad/` for platform usage and configuration.
